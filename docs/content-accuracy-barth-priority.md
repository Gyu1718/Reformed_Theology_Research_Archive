# 내용 정확도 검수 원칙 — 바르트 신정통주의 우선

검수 기준 수정일: 2026-06-18

## 기본 방향

이 아카이브의 신학적 우선순위는 바르트의 신정통주의다. 따라서 주제 비교와 책 설명은 단순히 개혁파 정통과 바르트를 균형 있게 병렬하는 방식이 아니라, 바르트의 계시 중심·그리스도 중심 교의학을 해석의 기준점으로 삼고 개혁파 정통을 비교 대상으로 배치하는 방식으로 정리한다.

다만 바르트 우선이라는 말은 개혁파 정통을 일부러 부정확하게 설명한다는 뜻이 아니다. 비교 대상의 입장을 정확히 세워야 바르트의 차이도 선명해진다.

## 검수 원칙

1. 바르트 항목은 먼저 말한다.
   - 계시론, 성경론, 신학방법론, 자연신학, 예정론처럼 바르트의 전환이 결정적인 주제는 바르트 입장을 먼저 제시한다.
   - 문장 구조도 가능하면 “바르트는 … 이에 비해 개혁파 정통은 …”의 순서를 따른다.

2. 바르트의 핵심 축을 흐리지 않는다.
   - 계시: 인간 종교 경험이나 일반 원리가 아니라 예수 그리스도 안에서 일어나는 하나님의 자유로운 자기계시 사건.
   - 성경: 계시 자체와 단순 동일시되지 않지만, 하나님의 계시를 증언하도록 선택된 정경적 증언.
   - 신학방법론: 교의학은 교회 선포가 하나님의 말씀 앞에서 받는 자기검토.
   - 자연신학: 그리스도 밖의 독립적 자연신학과 자율적 접촉점 거부.
   - 예정론: 예수 그리스도는 선택하시는 하나님이자 선택받은 인간.
   - 신론: 하나님은 사랑 안에서 자유롭고 자유 안에서 사랑하시는 분.

3. 개혁파 정통은 비교 대상으로 정확히 설명한다.
   - 개혁파를 단순 명제주의, 추상 작정론, 자연신학 낙관론으로 축소하지 않는다.
   - 성경 권위, 성령의 내적 증거, 일반계시의 실재와 한계, 예정론의 은혜와 확신의 맥락을 보존한다.
   - 그래야 바르트의 문제의식이 무엇을 비판하고 재배열하는지 정확히 보인다.

4. 비교 문장은 대립만 만들지 말고 전환점을 드러낸다.
   - 나쁜 방식: “개혁파는 성경 명제, 바르트는 계시 사건.”
   - 좋은 방식: “바르트는 계시를 예수 그리스도 안의 하나님의 자유로운 자기계시 사건으로 우선 이해한다. 이에 비해 개혁파 정통은 성경에 기록되고 성령이 조명하는 하나님의 규범적 말씀을 계시 인식의 표준으로 강조한다.”

5. 렌더링 패치 적용 원칙
   - 당장 화면에 노출되는 비교 문장은 `js/content-accuracy-patch.js`에서 먼저 교정한다.
   - 장기적으로는 `data/topics.json` 원본에 반영한다.
   - 바르트 우선순위가 필요한 문장은 `diverge`와 `positions`에 우선 반영한다.

## 1차 반영 항목

`js/content-accuracy-patch.js`에 다음 항목을 바르트 우선 구조로 재정렬했다.

- 계시론
- 성경론
- 신학방법론
- 자연신학
- 신론
- 삼위일체론
- 예정론

## 다음 검수 적용 방식

다음 범위인 창조와 언약, 섭리와 악, 인간론과 죄론, 기독론, 속죄와 화해도 같은 원칙을 따른다.

- 먼저 바르트가 해당 주제를 어떻게 재배열하는지 확인한다.
- 그다음 개혁파 정통 설명이 바르트와 비교될 수 있을 만큼 정확한지 확인한다.
- 최종 문장은 바르트의 문제의식을 먼저 세우고, 개혁파 정통을 비교 위치에 둔다.
