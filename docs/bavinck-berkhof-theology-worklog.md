# Bavinck & Berkhof Theology Index Worklog

## 작업 방향

바르트 책작업과 같은 방식으로 바빙크와 벌코프를 책 상세 페이지에서 읽을 수 있는 연구 색인으로 확장한다. 원문 전문을 넣지 않고, 구조·요약·소주제·학습 질문·비교 메모 중심으로 구성한다.

## 1차 적용 — 바빙크

- 바빙크 항목을 업로드된 『개혁교의학 개요』 EPUB 목차 기준 24장 구조로 조정했다.
- 기존 4권식 추상 구조를 5개 연구 단위로 재배열했다.
  - 인간의 최고선과 계시의 원리
  - 하나님, 창조, 인간
  - 죄, 언약, 그리스도
  - 성령과 구원의 적용
  - 교회와 영생
- `data/books/bavinck-chapter-profiles.json`를 추가해 장별 요약, 소주제, 핵심 포인트, 개념 태그를 분리했다.
- `js/bavinck-preload.js`가 구조 맵과 장별 프로필을 함께 읽도록 업데이트했다.

## 2차 적용 — 벌코프

- 벌코프 『조직신학』 EPUB의 `toc.ncx`를 기준으로 상세 목차 구조를 확인했다.
- 기존 39장 압축 구조는 유지하고, 책 상세 페이지 하단에 별도 `EPUB 상세 목차 색인`을 추가했다.
- `js/berkhof-detailed-toc-preload.js`를 추가해 벌코프 상세 페이지에 7개 교리 영역, 26개 중간 단위, 81개 세부 단위를 접기/펼치기 UI로 표시하도록 했다.
- `index.html`에 해당 프리로드 스크립트를 연결했다.
- 원문 전문은 포함하지 않고, 목차 구조와 탐색용 제목만 수록한다.

## 벌코프 쪽 다음 작업

다음 단계는 81개 세부 단위 아래에 있는 A/B/C식 하위 항목 332개를 별도 JSON 데이터로 분리하고, 각 단위에 2–4문장 설명·핵심 질문·비교 포인트를 붙이는 것이다. 이때 현재 `berkhof-chapter-profiles.json`와 중복되지 않도록, 기존 장별 프로필은 교리 요약용으로 유지하고 상세 목차 데이터는 탐색용으로 유지한다.
