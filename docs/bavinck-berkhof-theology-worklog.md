# Bavinck & Berkhof Theology Index Worklog

## 작업 방향

바르트 책작업과 같은 방식으로 바빙크와 벌코프를 책 상세 페이지에서 읽을 수 있는 연구 색인으로 확장한다. 원문 전문을 넣지 않고, 구조·요약·소주제·학습 질문·비교 메모 중심으로 구성한다.

## 1차 적용

- 바빙크 항목을 업로드된 『개혁교의학 개요』 EPUB 목차 기준 24장 구조로 조정했다.
- 기존 4권식 추상 구조를 5개 연구 단위로 재배열했다.
  - 인간의 최고선과 계시의 원리
  - 하나님, 창조, 인간
  - 죄, 언약, 그리스도
  - 성령과 구원의 적용
  - 교회와 영생
- `data/books/bavinck-chapter-profiles.json`를 추가해 장별 요약, 소주제, 핵심 포인트, 개념 태그를 분리했다.
- `js/bavinck-preload.js`가 구조 맵과 장별 프로필을 함께 읽도록 업데이트했다.

## 벌코프 쪽 다음 작업

현재 벌코프는 `data/books/berkhof-chapter-profiles.json`와 `js/berkhof-preload.js`를 통해 장별 프로필과 소주제 설명이 이미 연결되어 있다. 다음 단계는 실제 EPUB의 세부 목차 깊이에 맞춰 현재 39장 압축 구조를 더 세밀한 대목차/중목차 인덱스로 확장하는 것이다.

## 비교축

- 칼빈: 종교개혁 고전 문헌의 원형
- 바빙크: 유기적 개혁파 교의학의 개요형 종합
- 벌코프: 교과서형 조직신학의 표준 배열
- 바르트: 계시 중심·그리스도 중심의 신정통주의 재구성
