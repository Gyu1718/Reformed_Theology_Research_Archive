# Bavinck & Berkhof Theology Index Worklog

## 작업 방향

바르트 책작업과 같은 방식으로 바빙크와 벌코프를 책 상세 페이지에서 읽을 수 있는 연구 색인으로 확장한다. 원문 전문을 넣지 않고, 구조·요약·소주제·학습 질문·비교 메모 중심으로 구성한다.

## 1차 적용 — 바빙크

- 바빙크 항목을 업로드된 『개혁교의학 개요』 EPUB 목차 기준 24장 구조로 조정했다.
- 기존 4권식 추상 구조를 5개 연구 단위로 재배열했다.
- `data/books/bavinck-chapter-profiles.json`를 추가해 장별 요약, 소주제, 핵심 포인트, 개념 태그를 분리했다.
- `js/bavinck-preload.js`가 구조 맵과 장별 프로필을 함께 읽도록 업데이트했다.

## 2차 적용 — 벌코프 영어판 EPUB 기준 상세 목차

- 사용자가 새로 올린 영어판 벌코프 EPUB 기준으로 상세 목차 구조를 다시 확인했다.
- 이 파일은 별도 서론 대영역이 아니라 여섯 개 교리 영역으로 전개된다.
- `data/books/berkhof-detailed-toc.json`를 추가해 6개 교리 영역, 14개 중간 단위, 70개 장, 293개 A/B/C급 세부 표제를 분리 저장했다.
- 원문 전문은 포함하지 않고, 구조 탐색용 제목만 수록한다.

## 3차 적용 — 벌코프 상세 목차와 기존 설명 프로필 연결

- 벌코프에는 이미 `data/books/berkhof-chapter-profiles.json`에 39개 장별 설명, 소주제 설명, 핵심 질문, 개념 태그가 들어가 있다.
- 새로 설명을 중복 작성하지 않고, `data/books/berkhof-epub-profile-map.json`를 추가해 영어 EPUB 실제 70개 장을 기존 39개 한국어 설명 프로필에 연결했다.
- 연결 품질은 `strong`과 `partial`로 구분했다. 은혜언약, 보통 은혜, 신비적 연합, 천년왕국 견해처럼 기존 39장 구조에 독립 프로필이 없는 항목은 `partial`로 표시했다.
- `js/berkhof-detailed-toc-preload.js`는 이제 상세 목차 JSON, 프로필 매핑 JSON, 기존 장별 프로필 JSON을 함께 읽고, 벌코프 상세 목차의 각 장 아래에 연결된 기존 설명 요약과 핵심 포인트를 표시한다.

## 벌코프 쪽 다음 작업

다음 단계는 새 설명 대량 작성이 아니라 연결 품질 점검이다. `partial`로 표시된 항목만 별도 보강하거나, 기존 39장 프로필을 70장 체계로 점진적으로 확장하면 된다.
