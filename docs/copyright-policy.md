# Copyright Policy

## 1. 기본 원칙

이 저장소는 공개 저장소입니다. 따라서 저작권이 있는 책의 원문 전체를 저장하거나 배포하지 않습니다.

이 프로젝트의 목적은 원문 복제나 배포가 아니라, 연구를 위한 색인과 메모를 관리하는 것입니다.

## 2. 공개 가능한 자료

다음 자료는 공개 저장소에 올릴 수 있습니다.

- 책 제목
- 저자명
- 출판 정보
- 목차 수준의 정보
- 장별 요약
- 주제 태그
- 성경 본문 태그
- 학자 태그
- 짧은 인용
- 페이지 위치 또는 장절 위치
- 직접 작성한 연구 메모

## 3. 공개하지 않는 자료

다음 자료는 공개 저장소에 올리지 않습니다.

- PDF 원문
- EPUB 원문
- OCR TXT 전문
- 번역서 본문 전체
- 스캔 이미지
- 긴 인용문
- 상업 출판물의 실질적 대체물이 될 수 있는 자료

## 4. 원문 파일 관리

원문 파일은 다음 방식으로 관리합니다.

- 개인 컴퓨터의 로컬 폴더
- 외장 저장 장치
- 비공개 클라우드
- private repository

공개 저장소의 `.gitignore`는 원문 파일이 실수로 올라가지 않도록 다음 확장자를 차단합니다.

```txt
*.pdf
*.epub
*.txt
*.docx
*.hwp
*.hwpx
```

## 5. 인용 원칙

인용은 연구와 비평 목적에 필요한 최소한으로 제한합니다.

권장 형식:

```txt
저자, 『책 제목』, 장/절 또는 페이지, 핵심 요약 메모
```

짧은 인용을 넣을 경우:

```txt
"짧은 인용문" — 저자, 『책 제목』, p. 00
```

## 6. 운영 원칙

- 공개 사이트는 색인과 메모 중심으로 운영합니다.
- 원문 전문이 필요한 경우 개인 보관 파일을 직접 확인합니다.
- 공개 자료만으로 책을 대체할 수 있을 정도의 분량을 제공하지 않습니다.
- 저작권이 불분명한 자료는 공개하지 않습니다.
