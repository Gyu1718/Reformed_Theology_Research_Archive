# 개혁전통의 역사 파트 설계안

## 1. 목적

`개혁전통의 역사` 파트는 종교개혁 이후 개혁파, 장로교, 신앙고백, 개혁파 정통주의, 청교도, 신칼빈주의, 한국 장로교 수용사의 흐름을 연구 색인으로 정리하기 위한 영역입니다.

이 파트는 전체 교회사 서술을 목표로 하지 않습니다. 목적은 개혁신학의 주요 교리 주제, 학자, 문헌, 성경 본문, 연구 메모를 역사적 맥락 안에서 연결하는 것입니다.

## 2. 범위

### 포함하는 범위

- 종교개혁 이전 배경 중 개혁신학 이해에 필요한 요소
- 16세기 종교개혁과 개혁파 전통의 형성
- 장로교 전통과 장로회 정치의 형성
- 도르트 총회와 개혁파 구원론
- 웨스트민스터 총회와 장로교 신앙고백
- 개혁파 정통주의와 스콜라적 방법
- 청교도와 교리교육 전통
- 화란 개혁주의와 신칼빈주의
- 프린스턴 신학과 미국 장로교
- 한국 장로교와 개혁신학 수용사

### 제외하는 범위

- 초대교회사 전체
- 중세교회사 전체
- 동방교회사 전체
- 로마가톨릭 교회사 전체
- 현대교회사 전체
- 세계교회사 일반 연표

이 파트의 기준 문장은 다음과 같습니다.

> 개혁신학을 이해하는 데 필요한 교회사만 다룬다.

## 3. 1차 핵심 항목

초기 데이터는 다음 6개 항목으로 시작합니다.

| id | 제목 | 역할 |
|---|---|---|
| `history-index` | 개혁전통의 역사 | 전체 안내와 흐름도 |
| `reformation-to-reformed` | 종교개혁에서 개혁파 전통으로 | 루터파, 개혁파, 재세례파, 로마가톨릭 개혁의 구분 |
| `reformed-and-presbyterian` | 개혁파와 장로교는 어떻게 다른가 | 개혁파, 장로교, 칼빈주의 용어 구분 |
| `synod-of-dort` | 도르트 총회와 개혁파 구원론 | 예정론, 은혜론, 항론파 논쟁의 연결 |
| `westminster-assembly` | 웨스트민스터 총회와 장로교 신앙고백 | 신앙고백, 교회정치, 예배, 언약신학의 연결 |
| `reformed-orthodoxy` | 개혁파 정통주의란 무엇인가 | 정통주의와 스콜라주의에 대한 오해 정리 |

## 4. 페이지 공통 구조

각 역사 항목은 다음 순서로 정리합니다.

1. 한 문장 요약
2. 핵심 정의
3. 역사적 배경
4. 주요 인물
5. 주요 문헌
6. 핵심 신학 쟁점
7. 연결되는 교리 주제
8. 연결되는 책과 학자
9. 자주 생기는 오해
10. 연구 활용 메모

이 구조는 긴 교회사 서술보다 연구 색인과 상호 링크에 적합합니다.

## 5. 데이터 파일

역사 데이터는 다음 파일에서 관리합니다.

```txt
/data/tradition-history.json
```

역사 입력 양식은 다음 파일에서 관리합니다.

```txt
/templates/history-entry.template.json
```

## 6. 데이터 연결 원칙

역사 항목은 독립 문서가 아니라 기존 데이터와 연결되어야 합니다.

- `relatedTopics`: `data/topics.json`의 주제 태그와 연결
- `relatedBooks`: `data/books.json`의 책 id와 연결
- `relatedAuthors`: `data/authors.json`의 학자 id와 연결
- `relatedPassages`: `data/passages.json`의 성경 본문 id 또는 표기와 연결
- `researchUses`: 설교, 논문, 브런치 글쓰기, 교리교육에서의 활용 방향 기록

## 7. 작성 원칙

- 공개 저장소에 저작권 있는 원문 전문을 올리지 않습니다.
- 역사 설명은 직접 요약하여 작성합니다.
- 긴 인용문은 넣지 않고, 필요하면 인용 위치만 남깁니다.
- 특정 전통을 단순화하거나 과도하게 변증하지 않습니다.
- 용어 구분을 우선합니다. 예: 개혁파, 장로교, 칼빈주의, 정통주의.
- 각 항목은 반드시 교리 주제와 연결합니다.

## 8. 확장 예정 항목

1차 항목 이후에는 다음 항목을 확장할 수 있습니다.

- `puritanism` — 청교도와 개혁파 경건
- `dutch-reformed` — 화란 개혁주의
- `neo-calvinism` — 신칼빈주의와 일반은총
- `old-princeton` — 프린스턴 신학과 성경론
- `korean-presbyterianism` — 한국 장로교와 개혁신학 수용
- `barth-and-reformed` — 바르트와 개혁파 전통의 긴장
- `modern-reformed-theology` — 현대 개혁신학의 주요 흐름

## 9. 1차 작업 목표

이번 단계의 목표는 다음 세 가지입니다.

1. 역사 파트의 범위와 작성 규칙을 고정한다.
2. 입력 템플릿을 만든다.
3. 초기 6개 항목을 데이터 파일에 넣는다.

이후 작업은 `tradition-history.json`의 각 항목을 실제 화면에 렌더링하고, `topics.json`, `books.json`, `authors.json`와 상호 연결하는 방향으로 진행합니다.
